# How to run the ToDos App

# on Windows
1. open cmd
2. navigate to directory of ToDos App
3. "py manage.py runserver"
4. open http://127.0.0.1:8000/todos in Browser

# on Linux
1. open console
2. navigate to directory of ToDos App
3. "python manage.py runserver"
4. open http://127.0.0.1:8000/todos in Browser
